# Bio-Ambient Shield
Advanced Non-Invasive Vital Signs Monitoring System for Infants using UWB Radar and Edge AI.

## Overview
Bio-Ambient Shield is a next-generation monitoring solution designed to provide continuous, contact-free health tracking for infants. By utilizing UWB (Ultra-Wideband) radar technology and real-time Edge AI analysis, the system tracks vital signs without the need for wearable sensors, cameras, or microphones.

## Key Features
- **Contact-Free:** Radar-based detection of sub-millimeter chest movements.
- **Privacy First:** Zero images, zero audio – purely motion telemetry.
- **Edge AI:** Real-time processing via FFT and State Machine logic.
